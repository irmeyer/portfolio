Wordle Clone - JavaScript-Based Word Puzzle Game

Overview:

I developed a fully functional Wordle clone using JavaScript, HTML, and CSS, closely mimicking the mechanics of the New York Times Wordle game. This project showcases my ability to build an interactive and engaging web-based game from scratch, demonstrating my skills in front-end development, DOM manipulation, event handling, and game logic implementation.

Features:

  • Dynamic Game Board: Displays six rows of tiles for user guesses, with real-time updates and animations. Keyboard Input & On-Screen Keyboard: Players can enter words using their keyboard or the interactive on-screen keyboard.
  • Color-Coded Feedback: Letters turn green (correct & in the right place), yellow (correct but misplaced), or gray (incorrect), following the standard Wordle rules.
  • Word Validation: Ensures that only real words from a predefined list are accepted. Game State & Logic Handling: Tracks player progress, limits guesses to six attempts, and displays a notification upon winning or losing.
  • UI Enhancements: Includes animations like shaking incorrect words and bouncing correct answers for a polished experience.

Technologies Used:

  • JavaScript (ES6+)
  • HTML5 & CSS3 (including animations and responsive design)
