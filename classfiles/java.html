Thneed Inc. Order and Inventory Management System

Java Application for Order Tracking, Inventory Management, and Reporting

Overview:

I developed a comprehensive order and inventory management system for Thneed Inc., utilizing Java to streamline customer order tracking, manage inventory levels, and generate insightful business reports. The project features two phases: in Phase 1, I implemented a customer and order management system with a GUI that allows users to view, update, and store order data. In Phase 2, the system was extended to track inventory levels, estimate order fill dates, and provide detailed reports to optimize business operations.

Features:

  • Customer & Order Management: Users can add, view, and update customer and order data through an intuitive GUI.
  • Inventory Tracking: The system automatically updates inventory levels upon order completion and restocks, estimating fill dates for backordered items.
  • Real-Time Inventory Display: The GUI reflects current inventory by size and color and adjusts as stock is updated.
  • Report Generation: Generates detailed reports to analyze order fulfillment speed, inventory trends, and customer purchase behavior.
  • File I/O Integration: Supports saving and loading data to/from flat files to maintain persistent order and customer records.

    • Technologies Used:

    • Java (including Object-Oriented Programming concepts like classes and file I/O)
    • Swing (for GUI design)
    • JUnit (for unit testing and debugging)
